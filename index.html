About Me

Kevin Cho is a 4th year student at St. Edwards pursuing a Bachelor of Arts in Philosophy and certificate for The Coding Bootcamp at UT Austin in Full-Stack Web Development. After hearing about good prospects in the market and potential for collaboration from close friends, Kevin found his passion for Web Development. Kevin enjoys figuring out mechanical/technical problems when they occur as well as the logic behind coding, and found software languages to be a fit. Throughout the coding program, Kevin aims to achieve profiency in HTML, CSS, Javascript, React, Node.js, Bootstrap, MongoDB, PHP frameworks, MySQL, Heroku, Security and Session Storage, and User Authentication.

Served allows users to input current ingredients to generate recipes based on what they already have at home. Each recipe generated from their selected ingredients shows a thumbnail of the recipe, the percentage of which it matches the ingredients the user has listed and a heart button at the bottom of the card to "save" their favorite recipes for later use.
Saved recipes are reachable with the button in the upper right section of the page. There are also buttons in the navigation menu to see the developers and report bugs/issues with the site.

Served: a recipe generator powered by Spoonacular, a web API
